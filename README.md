# Reproducible Tabular Machine Learning

A proof-of-concept for reproducible tabular machine learning benchmarks that work with OpenML, scikit-learn, and arbitrary local datasets. It provides lightweight specifications for datasets, tasks, resampling, and evaluation.
This project is inspired by the work conducted at the CD Laboratory for Dependable Intelligent Systems in Harsh Environments at TU Graz.

## Requirements

* Python 3.11+
* `uv` or `Miniforge3-Linux-x86_64.sh`

## Setup

Install project dependencies:

```shell
uv sync --dev
```
